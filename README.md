# Eat-n-Split

A bill-splitting app. Add friends, enter a shared bill, and the app tracks who owes who.

Built while learning React — based on a course project.

## What I practiced
- Controlled forms with multiple inputs
- Derived state (calculating balances)
- Conditional rendering
- Lifting state and sharing data between components

## Tech
- React 18
- Create React App

## Run locally

```bash
npm install
npm start
```
